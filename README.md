# Swear Media v2

A bespoke Shopify theme designed for filmmakers, production houses, and creative agencies. This theme is built to showcase and merchandise video content, from feature films to service packages.

## Key Features

*   **Hero Section**: A large, customizable hero section to make a strong first impression.
*   **Featured Grid**: Showcase a curated selection of films or products on the homepage.
*   **Film & Collection Templates**: Specialized templates for showcasing films (`product.film.liquid`) and organizing them into collections (`collection.films.liquid`).
*   **Service Packages**: A dedicated template (`product.service-package.liquid`) for selling services like editing, color grading, or consultation.
*   **Instagram Integration**: An embed section to display your latest Instagram posts.
*   **Trend Report**: A section for sharing news, articles, or trend reports.
*   **Contact Form**: A simple, integrated contact form for inquiries.

## Project Structure

The theme follows Shopify's standard directory structure:

*   `/assets`: Contains all theme assets, such as CSS, JavaScript, and images.
*   `/config`: Contains the theme's configuration files, including the settings schema.
*   `/layout`: Contains the main theme layout file (`theme.liquid`).
*   `/locales`: Contains theme localization files.
*   `/sections`: Contains the individual sections that can be added, removed, and reordered on pages.
*   `/snippets`: Contains reusable snippets of code.
*   `/templates`: Contains the page templates for the store.

## Customization

Theme customization can be done through the Shopify Theme Editor. You can modify the theme settings, add or remove sections, and change the overall look and feel of the store. For more advanced customizations, you can edit the theme code directly.
